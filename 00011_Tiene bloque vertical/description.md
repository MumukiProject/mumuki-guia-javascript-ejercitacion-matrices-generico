> Definí una función `tieneBloqueVertical` que tome una matriz y devuelva si la misma tiene un bloque _vertical_ :arrow_down: de 3 o más ítems consecutivos idénticos .
>
```javascript
ム tieneBloqueVertical([
  [4, 2, 3],
  [1, 5, 3],
  [1, 2, 3],
])
true
>
ム tieneBloqueVertical([
  [1, 2, 3, 4],
  [7, 2, 9, 2],
  [1, 8, 4, 5],
]) 
false
```
